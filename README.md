# Tomato Leaf Disease Detection using Deep Learning

A deep learning-based image classification system that detects diseases in tomato leaves using Convolutional Neural Networks (CNN) built with TensorFlow and NumPy.

## Dataset

The dataset contains images of tomato leaves categorized into multiple classes(10)

### Features:
- Detects tomato leaf diseases from input images
- CNN-based image classification using TensorFlow
- Supports multiple disease categories
- Fast and lightweight prediction system
- Can be extended to real-time camera detection

### Requirements:
- Python 3.12.10
- Libraries:
  - `tensorflow`
  - `numpy`
  - `matplotlib`


### Installation

1. Clone this repository or download the files :- git clone https://github.com/DULAKSHANA404/Tomato-Leaf-Disease-Detection
2. Install the necessary dependencies :- pip install -r requirements.txt
